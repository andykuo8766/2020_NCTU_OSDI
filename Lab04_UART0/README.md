Lab 4 - UART0, PL011
==========================
This Lab does the same as Lab 3, but it prints the serial number on UART0. As such, it can be used easily with qemu.


How to run
==========================
```sh
$ make
$ make run
```

How to clean
==========================
```sh
$ make clean
```
